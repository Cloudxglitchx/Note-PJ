✨ All Features (8 Total) + AI!

🔐 User Authentication

Secure login system with student and teacher roles.

✓ Quick login buttons for demo accounts
✓ Create your own account (student or teacher)
✓ Secure logout functionality
✓ User name displayed in navbar
How to use: Click "Sign in as Demo Student" or create your own account on the login page.

📝 Google Docs-Style Editor

Full-page note editor for taking notes during class.

✓ Clean, distraction-free writing space
✓ Auto-save every 2 seconds
✓ Text formatting (bold, italic, underline)
✓ Word and character count
✓ Copy/paste from Google Docs
How to use: Click "Add Note" button, type your notes, click "Publish Note" when done.

📅 Lesson Calendar

Browse past lessons and notes organized by date.

✓ Monthly calendar view
✓ List view for chronological browsing
✓ Click dates to see lessons and notes
✓ Visual indicators for activity
✓ Navigate between months
How to use: Click "Calendar" in navigation, browse by month or switch to list view.

💬 Discussion Comments

Discuss notes, ask questions, and collaborate.

✓ Comment on any note
✓ See comment count on note cards
✓ User avatars and timestamps
✓ Delete your own comments
✓ Keyboard shortcut (Ctrl+Enter)
How to use: Click any note card, scroll to comments, type and post your comment.

⬆️ Vote on Notes

Upvote helpful notes to highlight quality content.

✓ Upvote helpful notes
✓ Downvote if needed
✓ See vote count on cards
✓ Sort by "Top Today"
✓ Top note badge for highest rated
How to use: Click ⬆️ or ⬇️ arrows on note cards to vote.

🔍 Search & Filter

Find notes quickly with search and filtering.

✓ Search by title or content
✓ Filter by "Top Today" or "Newest"
✓ Real-time search results
✓ Browse by lesson or subject
How to use: Use search box to find notes, click filter buttons to sort.

📎 File Uploads

Attach images, PDFs, and documents to notes.

✓ Drag and drop file upload
✓ Support for images, PDFs, documents
✓ Image preview thumbnails
✓ Download attached files
✓ Multiple files per note (Max 5MB each)
How to use: When creating a note, scroll to "Attach Files" and drag files or click to browse.

🤖 AI Study Tools NEW!

AI-powered features using Google Gemini to enhance learning.

✓ Summarize Notes - AI-generated summaries
✓ Generate Quizzes - Practice questions from notes
✓ Study Tips - Personalized recommendations
✓ Key Points - Extract main concepts
✓ FREE with Google Gemini API
Setup (2 minutes):
  1. Visit ai.google.dev
  2. Click "Get API Key" (FREE)
  3. Copy your API key
  4. Go to AI Settings
  5. Paste key and save
  6. AI features enabled! ✨

🚀 Quick Start Guide

For Students:

  1. 1 Login with demo account or create your own
  2. 2 Browse existing notes on the dashboard
  3. 3 Click "Add Note" to create your own notes
  4. 4 Click notes to read and add comments
  5. 5 Use calendar to browse by date
  6. 6 Vote on helpful notes

For Teachers:

  1. 1 Login with teacher account
  2. 2 Post lesson overviews for students
  3. 3 Review student notes and comments
  4. 4 Answer questions in comments
  5. 5 Track engagement via calendar
  6. 6 Encourage collaboration
